# docs/ — the shared design contract

The **one** docs home for a game — no scattered loose `*_plan.md` at the root, no
parallel `references/` + `todos/` doc piles (v1 had both). `brief.md` is the
shared design brief; `decisions/` records the durable design decisions agents and
humans agree on. This is the canonical, agreed contract; work-in-progress human
notes stay in `refs/notes/`, and task tracking lives in Trello, not in files here.

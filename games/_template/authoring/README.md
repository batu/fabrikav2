# authoring/

Optional home for a game's editor-native project authority. A game may keep a
GrapesJS project, Phaser Editor project, or another evaluated frontend here when
that project is the editable source that produces the runtime UI projection.

The template does not choose an editor, create a scaffold, or define a default
authoring workflow. Leave this directory documentation-only until the game
adopts an editor authority explicitly. Validation inventories and editor-specific
projects should remain clearly separated so this directory does not become a
second ambiguous presentation authority.

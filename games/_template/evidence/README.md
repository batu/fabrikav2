# evidence/ — HUMAN SEAM (promoted artifacts only)

Durable, **promoted** agent artifacts: screenshots, before/after comparisons,
demo reels that prove a change works or document how the game looks. One dated,
topic-named subdirectory per batch: `<date>-<topic>/` (e.g.
`2026-07-06-coin-fly-tuning/`). Nothing lands here automatically — an artifact is
copied in only when it is worth keeping. Disposable run output stays in `.work/`;
this directory is the curated record.

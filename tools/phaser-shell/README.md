# @fabrikav2/phaser-shell

Lane tool workspace for the Phaser Editor (`phaser-native`) lane of the
dual-design-frontends experiment. U1 preseeds this manifest with the accepted
U2 pin (phaser 4.2.1 exact; Phaser Editor 5.0.2 stays a human-authenticated
desktop app and never enters the repo). U5/U6 implement the authoring and
projection tooling. Lane fences: `experiments/design-frontends/fences.json`.

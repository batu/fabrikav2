Through-lines

Patterns that repeat across screens — fix once, help everywhere.

Wrong container chrome

Ours uses white-glow / beige borders; the reference uses saturated-blue outlines on every pill, card, and button. One theme fix touches every screen.

Missing wooden backdrop

Reference modals (settings, fail, win) sit on a brown wooden board layer; ours float on flat purple.

Modals over flat purple

Reference dims the live game behind each dialog; ours shows a solid scrim with nothing behind — a depth cue lost.

Settings & Fail are structural

Not restyles — wrong header, wrong buttons, missing emoji/copy. The two lowest scores and the two biggest available wins.
